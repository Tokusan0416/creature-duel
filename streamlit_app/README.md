# Creature Duel - Streamlit UI

Creature Duelのインタラクティブなウェブインターフェースです。

## 📦 インストール

```bash
# プロジェクトルートから
pip install -e ".[streamlit]"
```

## 🚀 起動方法

```bash
# プロジェクトルートから
streamlit run streamlit_app/app.py
```

ブラウザで http://localhost:8501 が自動的に開きます。

## 📂 ディレクトリ構造

```
streamlit_app/
├── app.py                      # メインアプリ（ホーム）
├── README.md                   # このファイル
│
├── pages/                      # ページ（自動的にサイドバーに表示）
│   ├── 1_🎮_battle.py         # バトル実行
│   ├── 2_📚_master_data.py    # マスタデータブラウザ
│   ├── 3_📜_battle_logs.py    # バトルログビューワー
│   └── 4_📊_analytics.py      # 統計・分析
│
└── utils/                      # ヘルパー関数
    ├── battle_runner.py       # バトル実行・ログ管理
    ├── formatters.py          # フォーマット関数
    └── visualization.py       # 可視化関数（Plotly）
```

## 💾 データ保存

バトルログは自動的に `battle_logs/` ディレクトリに保存されます：

```
battle_logs/
└── battle_20260330_120000_player1.json
```

ファイル名の形式：`battle_{日付}_{時刻}_{勝者}.json`

## 🎯 使い方

### 1. バトルを実行

1. 「🎮 Battle」ページに移動
2. Player 1とPlayer 2のクリーチャーを選択
3. 「バトル開始」ボタンをクリック
4. 結果とログを確認

### 2. マスタデータを確認

1. 「📚 Master Data」ページに移動
2. タブでクリーチャー・技・特性・タイプ相性を切り替え
3. 検索・フィルター機能で絞り込み
4. 詳細を展開して確認

### 3. バトルログを閲覧

1. 「📜 Battle Logs」ページに移動
2. ログ一覧から過去のバトルを選択
3. サマリー・ログ・JSONタブで詳細を確認
4. 必要に応じてJSONをダウンロード

### 4. 統計を分析

1. 「📊 Analytics」ページに移動
2. 全体統計を確認
3. クリーチャー別・タイプ別の勝率を分析
4. ターン数・ダメージの傾向を確認
5. CSVをエクスポートして保存

## 🔧 カスタマイズ

### ページの追加

`streamlit_app/pages/` ディレクトリに新しいPythonファイルを追加すると、
自動的にサイドバーに表示されます。

ファイル名の形式：`{番号}_{アイコン}_{名前}.py`

例：`5_⚔️_double_battle.py`

### スタイルの変更

各ページのCSSは `st.markdown()` で直接記述されています。
カスタムCSSを追加するには、各ファイルの先頭で定義してください。

### 可視化の追加

`streamlit_app/utils/visualization.py` に新しいグラフ関数を追加し、
各ページから呼び出してください。

## 🐛 トラブルシューティング

### Streamlitが起動しない

```bash
# 依存関係を再インストール
pip install -e ".[streamlit]" --force-reinstall
```

### モジュールが見つからない

```bash
# プロジェクトルートから起動していることを確認
cd /path/to/creature-duel
streamlit run streamlit_app/app.py
```

### バトルログが表示されない

- まず「🎮 Battle」ページでバトルを実行してください
- `battle_logs/` ディレクトリが自動的に作成されます

### グラフが表示されない

- Plotlyがインストールされているか確認
- ブラウザのJavaScriptが有効になっているか確認
